# 배포 전 체크리스트

## 파일 확인

- [ ] `app.py`가 저장소 최상단에 있음
- [ ] `requirements.txt`가 저장소 최상단에 있음
- [ ] `.streamlit/config.toml`이 포함되어 있음
- [ ] `school_master_final_v2.csv`가 포함되어 있거나, 앱 내부 데모 데이터로 실행 가능함

## 데이터 확인

- [ ] 실제 학생 개인정보가 들어 있지 않음
- [ ] 학교명 공개가 부담되면 가명 처리함
- [ ] 신청 사유, 긴급 사유 등 민감 문구는 요약·범주화함
- [ ] 숫자 컬럼에 쉼표나 원 단위 문자가 섞이지 않도록 정리함

## Streamlit Cloud 설정

- [ ] Repository 선택
- [ ] Branch 선택
- [ ] Main file path: `app.py`
- [ ] Deploy 클릭
- [ ] 배포 후 상단 메뉴 5개가 모두 열리는지 확인
- [ ] 결과 요약, 분포 현황, 학교 상세 보기 화면을 확인

## 대회 제출 자료 연결

- [ ] PPT에 서비스 URL 삽입
- [ ] 보고서에 사용 데이터 출처와 전처리 흐름 기재
- [ ] 시연 영상에 필터 변경, 시나리오 적용, 학교별 리포트 화면 포함
